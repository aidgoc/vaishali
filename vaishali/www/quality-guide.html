1. Overview

DSPL manufactures safety-critical crane electronics — Anti-Collision Devices, driver monitoring systems, wind speed indicators, and load moment indicators. A defective product in the field can cause equipment failure or endanger lives. Quality inspection at every stage is not optional.

  • Every incoming component must be inspected before entering production
  • Every assembly stage has in-process checkpoints
  • Every finished unit must pass a full functional test before shipping
  • Rejected items trigger investigation and feed into the CAPA system
Our products protect crane operators and site workers. A missed defect can cause serious injury. Never skip an inspection step, even under delivery pressure.

The full quality loop connects to the complaint and CAPA process documented at /complaint-guide.

3. Step 1: Incoming inspection

Inspect goods received from supplier

Who: Quality Inspector / Store Manager  |  When: Before accepting into Stores-DSPL
  1. When a Purchase Receipt is created, go to Stock > Quality Inspection > New
  2. Set Inspection Type to Incoming
  3. Link the Reference Type to Purchase Receipt and select the reference document
  4. Select the Item Code being inspected
  5. Perform the following checks:
CheckWhat to verifyMethod
Visual inspectionPhysical damage, packaging integrity, labellingVisual
Component specsPart number matches PO, correct rating/valueDatasheet comparison
Electrical testingResistance, capacitance, voltage rating (for electronic components)Multimeter / LCR meter
Quantity verificationReceived qty matches challan qtyPhysical count
Batch/lot verificationBatch number matches supplier certificateCertificate of conformity
  1. Add readings in the Readings table (specification, value, min/max, status)
  2. Set overall status: Accepted or Rejected
  3. Click Save, then Submit
If Rejected: quarantine the items in a separate location. Do not mix with accepted stock. Notify managers immediately. Create a Warranty Claim against the supplier for defective goods.

4. Step 2: In-process inspection

Inspect during manufacturing

Who: Production Supervisor / Quality Inspector  |  When: During assembly (Stock Entry from Work Order)
  1. Go to Stock > Quality Inspection > New
  2. Set Inspection Type to In Process
  3. Link the Reference Type to Stock Entry and select the manufacture entry
  4. Perform the following checks:
CheckWhat to verifyPass criteria
Assembly qualityAll components correctly placed and orientedMatches assembly drawing
Solder jointsNo cold joints, bridges, or missing solderIPC-A-610 Class 2 minimum
CalibrationSensor outputs within spec rangeWithin min/max tolerance
Firmware flashCorrect firmware version loadedVersion matches release notes
Power-on testBoard powers up, display initialisesNo errors on startup
  1. Record readings with min/max values for objective pass/fail
  2. Set status: Accepted (continue to next stage) or Rejected (rework or scrap)
  3. Click Save, then Submit
Link the Quality Inspection to the Work Order in the remarks field for traceability back to the production batch.

5. Step 3: Outgoing inspection

Final test before shipping to customer

Who: Quality Inspector / Production Head  |  When: Before creating Delivery Note
  1. Go to Stock > Quality Inspection > New
  2. Set Inspection Type to Outgoing
  3. Link the Reference Type to Delivery Note and select the document
  4. Run full functional tests per product type:
CheckACD / DRME-DASH / F-DashWWSI / MRTTPS
Sensor calibrationYes—Yes—
Wiring and connectionsYesYesYesYes
Display readingsYesYesYesYes
Firmware versionYesYesYesYes
Power supplyYesYesYesYes
Load cell conditionYes—Yes—
Camera / display unit—Yes——
  1. Perform calibration verification with reference weights (ACD, DRM, WWSI, MRT)
  2. Record all readings with min/max values
  3. Set status: Accepted (approve for dispatch) or Rejected (rework and re-test)
  4. Click Save, then Submit
No unit ships without an Accepted outgoing Quality Inspection. This is a hard gate — the Delivery Note should not be submitted until QI is Accepted.

7. Linking to CAPA

When Quality Inspection is Rejected

Who: Quality Inspector + Production Head  |  When: Immediately after rejection
  1. Investigate the root cause of the rejection
  2. If the failure is systematic (not a one-off), create a CAPA record
  3. In the CAPA, document:
CAPA sectionWhat to fill
Root Cause5-Why analysis of why the defect occurred
Corrective ActionWhat was done to fix the immediate issue (rework, replace, scrap)
Preventive ActionWhat will change to prevent recurrence (process, supplier, design)
OwnerPerson responsible for implementing the preventive action
Target DateDeadline for completing the preventive action
  1. Submit the CAPA and track until closure
  2. After 30 days, conduct an effectiveness review:
    • Has the same defect recurred?
    • Were the preventive actions implemented?
    • Set status to Verified Effective or Reopened

See the full CAPA process at /complaint-guide.

If the same failure occurs 3 or more times across different batches, escalate to the Managing Director for a design review and potential batch recall.

10. Tips and best practices

  • Always link QI to the source document — Purchase Receipt for incoming, Delivery Note for outgoing, Stock Entry for in-process. Unlinked QIs lose traceability.
  • Use min/max values in readings — objective pass/fail criteria eliminate subjective judgement and make audits straightforward
  • Rejected items: quarantine immediately — move to a separate physical location and tag with the QI number. Never mix with accepted stock.
  • Review QI rejection trends monthly — aggregate rejections by item, supplier, and failure type. This feeds directly into CAPA priorities.
  • Incoming inspection sampling — for high-volume, low-risk items (fasteners, packaging), inspect a sample per AQL tables. For critical components (PCBs, sensors, load cells), inspect 100%.
  • Calibrate test equipment — multimeters, LCR meters, and reference weights must be calibrated annually. Record calibration dates.
  • Photo documentation — photograph defects found during inspection and attach to the QI record. Visual evidence is invaluable for RCA.
  • Supplier scorecard — track incoming QI acceptance rate per supplier. Suppliers below 95% acceptance should be audited or replaced.
  • Do not ship under pressure — if a unit fails outgoing inspection, it goes back to production. Customer delivery dates are renegotiated, not quality standards.

# DSPL Field PWA — Test Suite

End-to-end tests against the deployed PWA at `https://dgoc.logstop.com/field`.

## Layout

```
tests/
├── runner.js          # Playwright driver — runs all cases, prints report
├── helpers.js         # Login, mock session, common assertions
├── cases/
│   ├── smoke.js       # Every screen loads, no JS errors
│   ├── navigation.js  # Back-button stack, deep links
│   ├── forms.js       # Apply leave, log expense, new lead
│   └── details.js     # Quotation/lead/customer detail render
└── screenshots/       # Generated by runner on failures
```

## Run

```bash
cd ~/vaishali/tests
node runner.js                  # full suite
node runner.js smoke            # only smoke tests
node runner.js forms.applyLeave # one specific case
```

## Subagents

- **pwa-tester** (`.claude/agents/pwa-tester.md`) — runs the suite, summarises, files bugs.
- **pwa-reviewer** (`.claude/agents/pwa-reviewer.md`) — reviews recent code changes against PWA conventions.
